Project: AI Co-Writing Fairy Tale App

Goal:
An interactive React application where user and AI write a fairy tale together.

Core features:
- User writes a paragraph
- AI continues
- Every 3 turns AI may suggest 3 plot choices
- User selects choice or continues writing
- Story history is stored locally

Tech:
- React + TypeScript
- Vite
- Zustand for state
- Tailwind for styling
- OpenAI API
